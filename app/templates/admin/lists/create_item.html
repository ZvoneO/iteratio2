About List Items

List items are the individual values that will appear in dropdown menus throughout the application.

Each item has:

  • Value: The text that appears in the dropdown
  • Description: Optional additional information
  • Order: Optional numeric value for sorting

Items with lower order values will appear first in dropdown menus.
