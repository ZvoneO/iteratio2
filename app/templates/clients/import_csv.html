File must be in CSV format with headers.
CSV Format Instructions
Required Format

Your CSV file should include the following columns:

  • name (required) - Client name
  • address (optional) - Street address
  • city (optional) - City
  • country (optional) - Country name
Example
name,address,city,country
Acme Inc.,123 Main St,New York,United States
TechCorp,456 Park Ave,San Francisco,United States
Notes
  • The first row must contain column headers
  • Clients with duplicate names will be skipped
  • Country names must match existing countries in the system
Large files may take some time to process.
